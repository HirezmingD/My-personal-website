01

项目背景与核心命题Background & core problem

中国县域经济中，土地一级市场由政府主导，但"这块地适合什么产业"的决策高度依赖人工经验。规划部门面对数十万宗地块和快速变化的产业政策，缺乏数据驱动的决策工具。另一方面，中小企业（尤其是制造业）在选址时面临信息不对称——他们不知道哪些县域、哪些地块符合自身需求，只能通过中介或关系网络低效获取信息。

In China's county-level economy, the primary land market is government-led, yet the decision of "which industry fits this parcel" remains heavily reliant on manual expertise. Planning departments face hundreds of thousands of parcels and rapidly shifting industrial policies without data-driven tools. Meanwhile, SMEs face information asymmetry in site selection — they lack efficient ways to identify suitable counties and parcels matching their needs.

本系统让 AI Agent 在每次评估中自进化，逐渐逼近最优的"地—产"匹配逻辑。政府端可获得全县产业最优布局建议，企业端可按需获取最优用地匹配——一个系统，两类用户，三项核心能力（规划评估、产业招商、企业择地）。

This system lets the AI agent self-evolve with each evaluation, progressively approaching optimal land-industry matching logic. Government users receive county-wide industrial layout recommendations; enterprise users get optimal site matches — one system, two user roles, three core capabilities (planning, investment, site selection).

Data foundation: 185,564 100m×100m grid cells covering Tonglu County, with CGCS2000→WGS84 coordinate transformation. The 74-field evaluation system spans traffic accessibility, industrial agglomeration, tenure coordination, facility proximity, and more. An R-tree spatial index powers an L0-L7 eight-level LOD pyramid with lasso-query responses <200ms. The enterprise database covers 14 industries including precision manufacturing, biopharma, logistics, and the digital economy.

自进化机制Self-evolution mechanism

本系统集成 EvoMap 自进化协议，让 Agent 的产业评估方法论随每次评估逐步优化。核心流程如下：

The system integrates the EvoMap self-evolution protocol, allowing the agent's industrial assessment methodology to improve with each evaluation:

  1. 提取经验基因Extract GeneGene
    每次评估完成后，自动提取七维权重分布与产业-地类匹配偏好，形成经验基因。After each evaluation, automatically extract seven-dimensional weight distributions and industry-land type matching preferences as an experience gene.
  2. 打包经验胶囊Package CapsuleCapsule
    Gene 附带场景描述、触发原因与影响摘要，打包为经验胶囊，支持去重与震荡保护。Gene is packaged with scenario description, trigger reason, and impact summary into a capsule, with deduplication and oscillation protection.
  3. 方法论注入Methodology injectionInject
    下次评估时，进化后的方法论自动注入 System Prompt——Agent 的知识在持续累积。Evolved methodology is automatically injected into the System Prompt for the next evaluation — the agent's knowledge accumulates continuously.
  4. 版本演进Version evolutionVersion
    反馈修正触发主版本升级；权重变化 >5% 触发次版本迭代；1 小时震荡保护防频繁跳动。Feedback corrections trigger major version upgrades; weight changes >5% trigger minor iterations; 1-hour oscillation protection prevents frequent flips.

隐私方案：公私分离Privacy: public-private separation

敏感决策数据（原始土地利用矢量、完整指标字段）永久留在本地 SQLite。通过 EvoMap 协议发布的只有脱敏后的评估方法论（Gene/Capsule）——数据不出域，经验可共享。这一设计让县域政府可以放心使用本地敏感数据，同时接入全国产业评估的知识网络。

Sensitive decision data (raw land-use vectors, full indicator fields) stay permanently in local SQLite. Only de-identified assessment methodology (Gene/Capsule) is shared via the EvoMap protocol — data stays local, experience travels. This design lets county governments safely use sensitive local data while tapping into a nationwide network of industrial assessment knowledge.
