第 0 步|先在腾讯云控制台放行端口
安全组入站规则加一条:允许 TCP 80(来源 0.0.0.0/0)。这步你会操作。
第 1 步|本机把三个文件传上去
先把上面三个文件下载到你本地电脑,然后在本地终端执行(把 <IP> 换成你的公网 IP;登录用户 Ubuntu 镜像通常是 ubuntu,也可能是 root):
bashscp index.html portfolio.pdf project-list.pdf ubuntu@<IP>:~/
第 2 步|登录服务器,装 Nginx
bashssh ubuntu@<IP>
sudo apt update && sudo apt install -y nginx
sudo systemctl enable --now nginx
装完先别急着传,浏览器打开 http://<IP>,看到 Nginx 的欢迎页 = 端口和防火墙都通了(先验证再往下,出问题好定位)。
第 3 步|把站点放进网站根目录
bashsudo rm -f /var/www/html/index.nginx-debian.html      # 删掉默认页
sudo cp ~/index.html ~/portfolio.pdf ~/project-list.pdf /var/www/html/
完成。再开 http://<IP>,就是你的主页了,http://<IP>/portfolio.pdf 也能直接下。静态站,以后改内容只要重新 scp 覆盖 index.html 即可,不用重启 Nginx。
—
如果你的机器是 TencentOS / CentOS(不是 Ubuntu),只有两处不同:
bashsudo dnf install -y nginx                              # 装
sudo cp ~/index.html ~/portfolio.pdf ~/project-list.pdf /usr/share/nginx/html/   # 根目录换成这个
万一 http://<IP> 打不开,九成是防火墙(不是安全组)。Ubuntu 的 ufw 默认是关的,一般不用管;若开着:sudo ufw allow 80/tcp。CentOS 用 firewalld:sudo firewall-cmd --permanent --add-service=http && sudo firewall-cmd --reload。
